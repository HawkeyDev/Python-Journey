# Day 12 - Number to Words

A terminal convertor for number to words in 0-9.

## What it does
- A dictonary called numbers is created with 0-9 as key and in words in value
- now the user give the input between 0-9.
- The output prints the number in words
- if the number is out of 0-9 then it prints out of range
  
## What I learned
- Creating a dictionary
- user input
- if/else condition to print the dictionary values or running the code without error
- f-strings — f"The number {num} is {numbers[num]}" to put variables inside strings

## Run it
python python.py
