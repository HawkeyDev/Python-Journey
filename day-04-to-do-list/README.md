# Day 04 - To-Do List App

A terminal task manager that tracks pending and completed tasks.

## What it does
- Add a task (saved as pending)
- Finish a task (status changes to done)
- Show all tasks with their status
- Data saved to tasks.json — persists after closing

## What I learned
- Storing task + status together in a dictionary
- Updating a value in a dictionary
- Checking if key exists before updating
- Same JSON load/save pattern as contact book
- Getting faster at while loop + menu structure

## Run it
python python.py
