
# Day 01 - Number Guessing Game

A simple terminal game built on day 1 of my Python journey.

## What it does
- Computer picks a random number between 1 and 100
- You get 7 chances to guess it
- Tells you if your guess is too high or too low
- Shows how many tries it took to get it right

## What I learned
- random module
- for loops
- if/elif conditions
- break statement
- taking user input

## Run it
python game.py
