# Day 03 - Contact Book

A terminal contact manager that saves contacts permanently.

## What it does
- Add a contact (name + phone number)
- Delete a contact by name
- Search a contact by name
- View all saved contacts
- Data saved to contacts.json — persists after closing

## What I learned
- JSON — saving and loading data to file
- json.dump and json.load
- Dictionaries and dict.items()
- dict.pop() for safe deletion
- while True loop with menu
- with open() file handling

## Run it
python python.py
