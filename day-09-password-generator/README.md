# Day 09 - Password Generator 
A simple terminal password generator using string libraries.
## What it does
- user selects the length of the password
- Computer picks password from letters(A-Z,a-z),digits(0-9),character(:;"'...)
- prints the password
## What I learned
- random module
- for loops
- string libraries (string.ascii_letters, string.digits, string.punctuation)
- taking user input
## Run it
python python.py
